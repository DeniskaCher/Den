## Клиент для отправки почты "Email client"

Автор: Черкасов Денис(КНФТКБ-101)

### Описание

Данное приложение служит для отправки почты по протоколу SMTP. На 22.04.18 находится на начальной стадии разработки с ограниченным функционалом, доступна возможность отправки самого письма с указанием темы и прикреплением вложений. В будущем можно будет отправлять письмо нескольким адресатам, plain text-письма и т.д

### Требования

+ Python версии не ниже 3
+ Последняя версия Tkinter

### Состав

+ Графическая версия: smtp.py
+ Тесты: tests.py
+ Изображение окна: icon.png
+ Примеры файлов для вложений: example.txt, vernik.jpg

### Графическая версия

+ Пример запуска: py smtp.py

### Подробности реализации

Графический интерфейс реализован с помощью библиотеки Tkinter.
